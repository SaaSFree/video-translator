# Frontend V1 页面结构图与组件清单

## 1. 页面结构图

```text
页面根节点
├── 左侧 Sidebar
│   ├── 模型设置区
│   │   ├── 语音转写下拉
│   │   └── Codex纠错下拉
│   ├── 当前任务状态区
│   │   ├── 第一层状态
│   │   ├── 第二层状态
│   │   └── 辅助信息
│   └── 项目浏览区
│       ├── 项目列表
│       └── 导入视频按钮
└── 右侧 Workspace
    └── Source 工作台
        ├── 原视频播放器
        ├── 播放控制条
        ├── 纠错摘要栏
        └── Source 分段列表
            └── 分段卡片（重复）
                ├── 段头信息
                ├── 正文文本框
                ├── 段操作按钮
                └── 纠错建议卡（可选）
```

## 2. 组件清单

### 2.1 Sidebar 级组件

1. `model-settings`
   - 作用：展示运行时模型配置
   - 输入：系统状态接口返回的 `settings` 与 `options`
   - 输出：更新系统运行时设置

2. `job-status-rail`
   - 作用：展示当前项目处理状态
   - 输入：项目详情中的 `job`、`manifest`、`source_correction_review`
   - 输出：无

3. `project-list`
   - 作用：浏览、切换、右键操作项目
   - 输入：项目列表与当前项目 id
   - 输出：切换项目、触发菜单动作

4. `file-input`
   - 作用：导入 mp4 视频
   - 输入：本地文件
   - 输出：创建项目并上传原视频

### 2.2 Workspace 级组件

1. `source-video`
   - 作用：播放原视频
   - 输入：项目 `source_video`
   - 输出：播放状态

2. `controls`
   - 子项：
     - 播放
     - 暂停
     - 停止
     - 重放
     - 倍速
     - 音量
     - source 语言
     - `转写分段`

3. `source-review-toolbar`
   - 作用：展示纠错整体状态与“全部待确认项”开关
   - 输入：纠错 review 文档
   - 输出：批量作用模式开关

4. `source-segments`
   - 作用：承载全部 source 段卡片
   - 输入：`source_segments`
   - 输出：段内按钮事件

### 2.3 分段卡片组件

每张卡片至少由以下部分组成：

1. `segment-head`
   - 段号
   - 时间范围
   - 正文状态标签

2. `segment-text`
   - 当前正文
   - 支持人工直接改写

3. `segment-actions`
   - `播放片段`
   - `定位视频`
   - `保存正文`

4. `review-card`
   - 仅在显示纠错时出现
   - 包含：
     - 建议状态
     - 原文
     - 建议
     - 结构化修改项
     - `接受`
     - `拒绝`

## 3. 右键菜单组件

### 3.1 项目右键菜单

菜单项固定为：

1. `停止任务`
2. `导出项目`
3. `显示纠错`
4. `隐藏纠错`
5. `截取片段`
6. `清空项目`
7. `删除项目`

### 3.2 文本右键菜单

文本框右键菜单固定为：

1. `全选`
2. `复制`

## 4. 弹窗组件

### 4.1 截取片段弹窗

字段：

1. 开始时间
2. 结束时间

按钮：

1. 取消
2. 确认截取

用途：

- 基于当前项目快速裁出一个新的片段项目

## 5. 组件边界

当前组件边界有两条核心约束：

1. source 工作台只处理 source 数据，不接触 target 结果。
2. 纠错建议只作为“建议层”，不能直接覆盖正文，必须通过接受、拒绝或自定义保存进入正文。

## 6. 已删除的旧组件

本版已移除以下旧组件：

- target 视频播放器
- target 文本列表
- 文本翻译模型设置
- 语音合成模型设置
- 完整流程按钮
- 翻译合成按钮
- source / target 双向同步滚动
